# Go simple CRUD API

This repo demonstrate on how you can create APIs and backend services with go.

## Installation

Fork the repo

install [gorilla/mux](https://github.com/gorilla/mux)
```bash
go mod init <project name>
go get -u github.com/gorilla/mux
```

## Usage

```bash
go build
go run main.go
```
